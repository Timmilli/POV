## Bluetooth commands

### Time commands

To get the current displayed time: `get`

To set the time to the value hh:mm:ss: `setHHMMSS`

### Change mode command

To change the current mode: `modXXX` where XXX can be:
- `CLS` for a classic clock
- `STR` for a straight clock
- `IMG` for an image

### Other command

To get the current speed: `spd`
