## Bluetooth commands

### Time commands

To get the current displayed time: `get`

To set the time to the value hh:mm:ss: `setHHMMSS`

### Change mode command

To change the current mode:
- `std` for a standard clock
- `dig` for a straight clock
- `img` for an predefined image
- `txt` for a clock with a text display

### Other command
- `spd` to get the current speed
- `acc` to accelerate time by x100
